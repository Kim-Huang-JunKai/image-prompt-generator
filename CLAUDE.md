# AI 生图提示词生成器

**触发词：** 生成提示词、生图提示词、AI 绘画、prompt、image prompt

本技能帮助用户为 AI 生图工具（豆包、Midjourney、Stable Diffusion 等）生成专业提示词。

---

## 🎨 可用风格

| 风格 ID | 风格名称 | 描述 |
|---------|---------|------|
| `3d-ecommerce-promotion` | 3D 电商促销插画 | C4D 渲染，粘土质感，天蓝色背景，活泼科技感 |
| `chibi-anime-product` | Q 版二次元角色产品展示 | 日系 Q 版动漫，赛璐璐上色，彩虹色系 |
| `3d-typography-promotion` | 3D 立体字促销海报 | 膨胀立体字，深蓝背景 + 暖色字，聚光灯效果，可爱 IP 点缀 |
| `chibi-professional-promo` | Q 版职业角色宣传 | Q 版职业人物，虚化工作室背景，大字标题，亲切专业感 |
| `handdrawn-graffiti-logo` | 手绘涂鸦字体 Logo | 手绘涂鸦立体字，黑白橙对比，拟人化工具，创意活泼 |

---

## 📋 交互流程

### 1. 首先展示风格选项
用户触发技能后，展示风格列表供选择。

### 2. 用户提供文案
根据选择的风格，提供对应文案模板。

### 3. 确认画幅
- `1:1` - 社交媒体封面、商品主图（推荐）
- `9:16` - 手机竖版海报、抖音/快手封面
- `16:9` - 横版广告横幅、网站 Banner

### 4. 生成提示词
输出完整的正面提示词、负面提示词、画幅比例和使用建议。

## 🎯 风格详情

### 3D 电商促销插画 (`3d-ecommerce-promotion`)

**核心特点：**
- 渲染：C4D 建模 + Octane 渲染器
- 质感：粘土质感 (Claysmorphism)，柔和哑光
- 配色：天蓝色背景 + 暖黄色功能区块
- 氛围：活泼科技感，商业促销

**文案模板：**
```
【产品/服务名称】[填写产品或服务名称]
【核心卖点】[1-3 个主要卖点，用逗号分隔]
【价格信息】[可选，如：特惠 50 元/台]
【服务地区】[可选，如：玉林玉州区]
【功能特性】[支持的平台/功能，用逗号分隔]
【备注说明】[可选，其他需要说明的信息]
```

**文案元素 → 视觉元素映射：**
| 关键词类型 | 视觉元素 |
|-----------|---------|
| 快速/速度 | 火箭、闪电、速度线 |
| 智能/AI/科技 | 机器人、大脑、芯片 |
| 安全/保护 | 盾牌、锁、防护罩 |
| 连接/社交 | 对话气泡、网络节点 |
| 优惠/促销 | 金币、礼盒、彩带、爆炸贴 |
| 物流/配送 | 箱子、翅膀、卡车、地图标记 |
| 数据/分析 | 图表、仪表盘、文档 |
| 创意/灵感 | 灯泡、云朵、彩虹 |
| 本地/地区 | 地图标记、定位图标 |
| 多平台/集成 | 多个应用图标组合 |

### 3D 立体字促销海报 (`3d-typography-promotion`)

- 氛围：促销感强，活泼热闹

```
【备注说明】[可选，其他特殊要求]
```

## ⚠️ 注意事项

1. **文字处理**：AI 生图工具文字渲染能力有限，具体文字需后期添加
2. **品牌图标**：知名品牌图标可能需要后期合成
3. **画幅选择**：根据发布平台选择合适画幅
4. **提示词长度**：建议控制在 800-1500 字符
