# 3D 电商促销插画风格

## 风格 ID
`3d-ecommerce-promotion`

## 核心美学

### 渲染引擎
- C4D 风格建模
- Octane 渲染器
- 8K 超高清细节

### 材质质感
- 粘土质感 (Claysmorphism)
- 柔和哑光表面
- 次表面散射效果 (SSS)

### 照明系统
- 全局柔光照明
- 边缘高光反射
- 体积光
- 无硬阴影

### 配色方案
```json
{
  "primary_background": "天蓝色 (#87CEEB)",
  "accent_blocks": "暖黄色 (#FFD700)",
  "outline_color": "深棕色 (#5D4037)",
  "highlight_gradient": ["电光蓝 (#00BFFF)", "活力黄 (#FFFF00)"]
}
```

## 构图系统

### 支持画幅
| 比例 | 用途 |
|------|------|
| 1:1 | 社交媒体封面、商品主图 |
| 9:16 | 手机竖版海报、抖音/快手封面 |
| 16:9 | 横版广告横幅、网站 Banner |

### 模块化区域

#### 1. Header Banner
- 形状：圆角矩形
- 背景：金色渐变 + 深棕描边
- 装饰：工厂管道、发光灯泡
- 用途：主标题、品牌名称

#### 2. Central Focus
- 字体：glossy 气泡立体字
- 角色：白色 3D 机器人 (戴头盔/挥手)
- 漂浮物：金币、笑脸 emoji、文档图标、云朵大脑
- 用途：核心视觉焦点

#### 3. Feature Cards
- 布局：横向三栏
- 设计：圆角方块 + 暖黄底色
- 图标语义：
  - 火箭/闪电 → 速度/直连
  - 对话角色 → 社交/沟通
  - 机械臂 + 翅膀箱 → 自动化物流

## 视觉语言
- 艺术风格：波普艺术、数字商业插画
- 细节等级：8K 超高清
- 情绪基调：活泼科技感
- 商业意图：产品推广/APP 营销

## 适配规则

### 文字处理
- 所有文字区域留白或使用占位符
- 禁止生成具体字符
- 使用"文字占位区域"描述

### 图标定制
- 底部图标按语义替换
- 保持 3D 粘土质感统一
- 大小适中，清晰可识别

### 色彩约束
- 主背景：天蓝色系 (#87CEEB 及相近色)
- 功能区块：暖黄色系 (#FFD700 及相近色)
- 可使用渐变色增加层次

### 禁止元素
- ❌ 写实摄影
- ❌ 扁平 2D 设计
- ❌ 暗黑风格
- ❌ 手绘草图
- ❌ 真实人物照片
- ❌ 复杂文字内容

## 文案模板

### 模板使用说明

用户选择此风格后，提供以下文案模板供用户填写，或直接参考示例格式自行编写。

### 基础文案模板

```
【产品/服务名称】[填写产品或服务名称]
【核心卖点】[1-3 个主要卖点，用逗号分隔]
【价格信息】[可选，如：特惠 50 元/台]
【服务地区】[可选，如：玉林玉州区]
【功能特性】[支持的平台/功能，用逗号分隔]
【备注说明】[可选，其他需要说明的信息]
```

### 文案示例

**示例 1：软件安装服务**
```
【产品/服务名称】openclaw 龙虾包安装
【核心卖点】特惠价格，本地服务，多平台支持
【价格信息】50 元/台
【服务地区】玉林玉州区
【功能特性】飞书，企业微信，AI 大模型
【备注说明】大模型自备
```

**示例 2：电商产品促销**
```
【产品/服务名称】智能手环新品上市
【核心卖点】7 天续航，心率监测，防水设计
【价格信息】首发价 199 元
【服务地区】全国包邮
【功能特性】运动计步，睡眠监测，消息提醒
【备注说明】限时赠送表带
```

**示例 3：APP 推广**
```
【产品/服务名称】XX 外卖 APP
【核心卖点】快速配送，满减优惠，品质保证
【价格信息】新用户立减 15 元
【服务地区】全市配送
【功能特性】在线支付，订单跟踪，客服支持
【备注说明】首单免配送费
```

### 画幅选择

填写文案后，选择输出画幅：
- `1:1` - 社交媒体封面、商品主图（推荐）
- `9:16` - 手机竖版海报、抖音/快手封面
- `16:9` - 横版广告横幅、网站 Banner
