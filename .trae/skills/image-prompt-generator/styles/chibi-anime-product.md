# Q 版二次元角色产品展示风格

## 风格 ID
`chibi-anime-product`

## 核心美学

### 艺术风格
- 日系 Q 版动漫角色 (Chibi)
- 赛璐璐上色风格
- 数字绘画
- 可爱活泼氛围

### 线条特征
- 清晰黑色轮廓线
- 线条流畅圆润
- 无断线或粗糙边缘

### 上色技法
- 赛璐璐分层上色
- 柔和渐变过渡
- 高光点缀
- 无复杂阴影

## 构图系统

### 支持画幅
| 比例 | 用途 |
|------|------|
| 1:1 | 社交媒体头像、商品主图 |
| 9:16 | 手机壁纸、竖版海报 |
| 4:3 | 传统插画展示 |

### 模块化区域

#### 1. 角色区域 (Central Character)
- 位置：画面中央或黄金分割点
- 比例：Q 版 2-3 头身
- 表情：开心笑脸/眯眼笑/眨眼
- 姿势：手指展示/挥手/比心
- 服装：彩虹色背带裙、可爱 T 恤

#### 2. 品牌展示板 (Brand Display)
- 位置：角色侧后方
- 形状：竖立矩形板
- 内容：品牌 Logo + 产品名称
- 装饰：彩虹星形闪光

#### 3. 产品展示台 (Product Table)
- 位置：画面前景下方
- 内容：相关产品陈列
  - 毛绒玩偶
  - 徽章/胸章
  - 钥匙扣
  - 产品包装盒
  - 托盘展示

#### 4. 对话气泡 (Speech Bubble)
- 位置：角色头部附近
- 形状：圆角气泡
- 内容：推销话语/产品介绍
- 风格：简洁可爱

## 视觉语言

### 角色特征
- **头部**：大头小身比例 (2-3 头身)
- **眼睛**：超大圆形眼睛，高光点缀
- **发型**：双马尾/单马尾/短发，配彩虹发饰
- **表情**：开心、活泼、元气满满
- **服装**：彩虹条纹背带裙、彩色 T 恤

### 色彩情绪
- 明亮欢快
- 马卡龙色系
- 彩虹元素
- 柔和温馨

### 细节等级
- 精致但不复杂
- 重点在角色表情
- 产品细节适度简化

## 适配规则

### 文字处理
- 对话气泡可包含简短文字
- 品牌展示板可显示品牌名称
- 产品包装可显示简化文字
- 避免过长文本

### 产品定制
- 根据实际产品调整展示台物品
- 保持与角色风格统一
- 产品比例适中清晰可辨

### 色彩约束
- 主背景：浅蓝色/浅粉色/浅紫色等马卡龙色
- 角色服装：可使用彩虹色或单色 + 彩色配饰
- 整体色调明亮柔和

### 禁止元素
- ❌ 写实风格
- ❌ 暗黑/恐怖元素
- ❌ 复杂背景
- ❌ 成人化内容
- ❌ 3D 渲染质感
- ❌ 扁平无轮廓设计

## 文案模板

### 基础文案模板

```
【角色设定】[角色描述，如：双马尾元气少女]
【品牌/产品名称】[填写品牌或产品名称]
【推销话语】[简短可爱的推销语，如：我卖这个、超好用的！]
【产品类型】[展示的产品类型，如：周边、玩偶、徽章等]
【配色偏好】[可选，如：彩虹色、粉色系、蓝色系]
【备注说明】[可选，其他特殊要求]
```

```
【推销话语】我卖这个
```

**示例 3：手作店铺**
