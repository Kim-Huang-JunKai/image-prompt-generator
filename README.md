# Image Prompt Generator

AI 生图提示词生成器 - 用于生成 AI 生图工具（如豆包、Midjourney、Stable Diffusion）的提示词。

## 🚀 使用方法

### Claude Code 项目级技能

本项目包含一个 Claude Code 项目级技能 (`CLAUDE.md`)，在 Claude Code 中打开此项目即可自动加载。

**触发方式：**
- "生成一个商品图片提示词"
- "用 3D 电商风格生成提示词"
- "帮我写一个 AI 绘画 prompt"

### 交互式流程

1. **选择风格** - 从可用风格列表中选择
2. **填写文案** - 提供产品/服务信息
3. **选择画幅** - 根据发布平台选择合适比例
4. **生成提示词** - 获得完整的正面和负面提示词

## 🎨 可用风格

### 1. 3D 电商促销插画 (`3d-ecommerce-promotion`)
- **特点**: C4D 渲染，粘土质感，天蓝色背景，活泼科技感
- **适用**: 商品推广、APP 营销、活动海报

### 2. Q 版二次元角色产品展示 (`chibi-anime-product`)
- **特点**: 日系 Q 版动漫角色，赛璐璐上色，彩虹色系
- **适用**: 产品推广、周边展示、社交媒体

## 📝 示例

### 3D 电商风格示例

**输入文案**:
```
【产品/服务名称】智能手环
【核心卖点】7 天续航，心率监测，防水设计
【价格信息】首发价 199 元
```

**输出**: 完整的 AI 生图提示词，包含主体、背景、材质、光照、色彩等详细描述

## 🎨 画幅比例

- **1:1** - 社交媒体封面、商品主图
- **9:16** - 手机竖版海报、抖音/快手封面
- **16:9** - 横版广告横幅、网站 Banner

## 🛠️ 扩展

### 添加新风格

编辑 `CLAUDE.md` 文件，添加新的风格定义和提示词模板。

### 建议的风格类型

- 极简产品风
- 国潮风
- 赛博朋克风
- 温馨手绘风
- 科技感蓝调
- 清新自然风

## ⚠️ 注意事项

1. **文字处理**: AI 生图工具文字渲染能力有限，具体文字需后期添加
2. **品牌图标**: 知名品牌图标可能需要后期合成
3. **画幅选择**: 根据发布平台选择合适画幅
4. **提示词长度**: 建议控制在 800-1500 字符
