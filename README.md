# AI 生图提示词生成器 - Claude Code 技能

一个用于生成 AI 生图提示词的 Claude Code 技能。

## 🎨 可用风格

| 风格 | 描述 |
|------|------|
| 3D 电商促销插画 | C4D 渲染，粘土质感，活泼科技感 |
| Q 版二次元角色 | 日系 Q 版动漫，赛璐璐上色 |
| 3D 立体字促销海报 | 膨胀立体字，聚光灯效果 |
| Q 版职业角色宣传 | 职业人物，工作室背景 |
| 手绘涂鸦字体 Logo | 涂鸦字体，拟人化工具 |

## 📦 安装方法

### 从 GitHub 安装

```bash
# 克隆仓库
git clone https://github.com/Kim-Huang-JunKai/image-prompt-generator.git

# 复制 skill.md 到你的项目
cp image-prompt-generator/skill.md /your/project/
```

### 直接使用

1. 下载 `skill.md` 文件
2. 将文件放入你的项目根目录
3. 在 Claude Code 中打开该项目

## 🚀 使用方法

在对话中使用以下指令触发技能：

- "生成提示词"
- "AI 生图提示词"
- "用 3D 电商风格生成"
- "手绘涂鸦风格，设计师接单"
