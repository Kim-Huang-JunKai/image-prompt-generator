# AI 生图提示词生成器

Claude Code 技能 - 为 AI 生图工具生成专业提示词。

## 🎨 可用风格

- **3D 电商促销插画** - C4D 渲染，粘土质感
- **Q 版二次元角色** - 日系 Q 版，赛璐璐上色
- **3D 立体字促销海报** - 膨胀立体字，聚光灯效果
- **Q 版职业角色宣传** - 职业人物，工作室背景
- **手绘涂鸦字体 Logo** - 涂鸦字体，拟人化工具

## 📦 安装方法

```bash
git clone https://github.com/Kim-Huang-JunKai/image-prompt-generator.git
```

将 `skill.md` 和 `styles/` 目录复制到你的项目即可使用。

## 🚀 使用方法

在 Claude Code 中使用：
- "生成提示词"
- "用 3D 电商风格生成"
- "手绘涂鸦风格，设计师接单"
