# 3D 电商促销插画 (`3d-ecommerce-promotion`)

## 核心特点

- **渲染**: C4D 建模 + Octane 渲染器
- **质感**: 粘土质感 (Claysmorphism)，柔和哑光
- **配色**: 天蓝色背景 + 暖黄色功能区块
- **氛围**: 活泼科技感，商业促销
- **适用**: 商品推广、APP 营销、活动海报

## 文案模板

```
【产品/服务名称】[填写产品或服务名称]
【核心卖点】[1-3 个主要卖点，用逗号分隔]
【价格信息】[可选，如：特惠 50 元/台]
【服务地区】[可选，如：玉林玉州区]
【功能特性】[支持的平台/功能，用逗号分隔]
【备注说明】[可选，其他需要说明的信息]
```
