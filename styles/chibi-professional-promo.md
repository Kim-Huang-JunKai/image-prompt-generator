# Q 版职业角色宣传 (`chibi-professional-promo`)

## 核心特点

- **角色**: Q 版职业人物（2-3 头身），大眼睛，温暖微笑
- **背景**: 虚化的室内工作室环境，浅景深，温暖自然光
- **文字**: 大号粗体字，白色描边 + 黑色投影
- **标签**: 圆角矩形信息标签（黄色底/白色底）
- **氛围**: 专业亲切，可信赖感
- **适用**: 服务接单、自由职业者宣传

## 文案模板

```
【职业/身份】[如：设计师、画师、剪辑师、程序员]
【主标题】[核心宣传语，如：自由设计师接单]
【副标题/卖点】[如：便宜可加急、可开票]
【价格信息】[可选，如：xx 元起]
【角色形象】[如：手持画笔、戴眼镜、拿相机]
【背景场景】[如：画室、工作室、办公室、咖啡馆]
【配色偏好】[可选，如：暖色调、蓝白、黄白]
【备注说明】[可选，其他特殊要求]
```
