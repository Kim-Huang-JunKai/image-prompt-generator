# 3D 立体字促销海报 (`3d-typography-promotion`)

## 核心特点

- **字体**: 3D 膨胀立体字，圆润饱满，充气效果
- **配色**: 深蓝色背景 + 橙色/桃色立体字
- **光影**: 顶部聚光灯照射，放射状光束
- **装饰**: 漂浮几何体、彩带碎片、可爱 IP
- **氛围**: 促销感强，活泼热闹
- **适用**: 闲鱼促销图、电商活动海报、直播封面

## 文案模板

```
【主标题】[核心宣传语，如：便宜可加急]
【副标题】[辅助说明，如：在闲鱼做任何设计]
【价格信息】[可选，如：20 元起]
【促销标语】[短句，如：专业高效、安全无忧]
【IP 偏好】[可选，如：小熊、小兔、小猫]
【配色偏好】[可选，如：深蓝 + 橙色、紫色 + 粉色]
【备注说明】[可选，其他特殊要求]
```
